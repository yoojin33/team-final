TEAM WEB-PEDIA

웹 지식을 정리하는 Web-Pedia 팀

Web-Pedia는 웹 개발 문법과 웹 보안을 한 곳에서 학습할 수 있도록 구성한 중간프로젝트입니다. 팀원별 링크는 GitHub, 블로그, 포트폴리오 주소를 공유해두었습니다.

프로젝트 개요

본 프로젝트는 웹 프로그래밍의 가장 기초가 되는 웹 표준 제작 문법을 체계적으로 정리하고, 전공자로서 반드시 숙지해야 할 주요 웹 보안 취약점과 해킹 방어 전략을 시각화하여 전시하는 데 목적이 있습니다.

  • 웹 표준 제작 문법: HTML/CSS 기초와 핵심 문법 소개
  • 웹 해킹 기법 및 원리: XSS, SQL Injection 등 대표 공격 원리 소개
  • 안전한 웹 코딩: 실무적인 보안 코딩 습관과 대응 방안

팀명 의미

Web-Pedia는 웹과 백과사전의 합성어입니다. 개발 문법부터 보안까지 흩어진 개념을 백과사전처럼 체계적으로 정리한다는 의미를 담았습니다.

프로젝트 주제

웹 개발 핵심 문법 및 웹 보안 취약점 소개 웹사이트
