프로젝트 구성

정보를 단순히 나열하지 않고, 문법 학습에서 보안 사고방식까지 이어지도록 구성했습니다.

01

웹 표준 제작 문법

HTML 문서 구조, 시맨틱 태그, CSS 선택자, 박스 모델, 반응형 레이아웃을 정리합니다.

  • HTML 기본 구조와 태그 역할
  • CSS 레이아웃과 스타일링 규칙
 02

웹 해킹 기법 및 원리

XSS, SQL Injection, CSRF 같은 공격이 어떤 흐름으로 발생하는지 시각적으로 이해합니다.

  • 공격 입력이 처리되는 과정
  • 취약 코드와 방어 코드 비교
 03

안전한 웹 코딩

입력 검증, 출력 인코딩, 인증/인가, 보안 헤더 등 실무적인 대응 습관을 체크합니다.

  • 보안 체크리스트
  • 안전한 구현 패턴
 04

학습 점검 퀴즈

HTML, CSS, JavaScript, 웹 보안 개념을 객관식 문제로 확인하고 결과를 바로 확인합니다.

  • JS로 채점되는 퀴즈
  • 틀린 문제를 카드에서 다시 복습

학습 흐름

초기 웹 제작 문법에서 시작해, 공격 원리와 방어 전략으로 자연스럽게 확장됩니다.

문서 구조 이해

HTML의 의미 있는 구조와 접근성, CSS 레이아웃의 기본 원리를 먼저 정리합니다.

취약점 원리 분석

STEP 3

방어 전략 적용

안전한 코딩 체크리스트로 실제 코드 작성 시 확인해야 할 항목을 점검합니다.
